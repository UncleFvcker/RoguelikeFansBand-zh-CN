<style:title>物品列表 (The Object List)</style>

此命令显示当前楼层上所有已知的物品。<color:keyword>Mogaminator (自动拾取器)</color> 想要拾取的物品会显示在靠近顶部的 <color:keyword>需要的物品 (wanted objects)</color> 类别中，而其他所有物品则显示在下方的 <color:keyword>其他物品 (other objects)</color> 类别中。这两组中的物品都按其估算价值进行排序；每件物品相对于你当前位置的位置，用类似 <color:keyword>N 23 E 11 (北23 东11)</color> 的方向格式来指示。物品列表还会显示地下城入口、任务入口和城镇商店，并且可以通过切换决定是否在列表中包含楼梯。

<topic:Summary><style:heading>快速汇总 (Quick Summary)</style>

  <color:keypress>Up/Down (上下方向键)</color>    一次滚动一行
  <color:keypress>PageUp/PageDown</color>         一次滚动一页
  <color:keypress>Home/End</color>                滚动到列表顶部/底部

  <color:keypress>/ 或 I</color>                  检视所选物品
  <color:keypress>J 或 ` 或 (</color>             旅行到所选物品
  <color:keypress>S</color>                       切换是否在列表中包含楼梯

  <color:keypress>ESC 或 Q</color>                退出此命令
  <color:keypress>?</color>                       显示此帮助主题

  <color:keypress>[a-z]</color>                   选择下一个匹配的条目

<style:note>注：由于现在按小写字母键会尝试匹配下一个首字母为该字母的条目，因此只能使用大写字母命令。例外情况是，对于 Roguelike 按键集的用户，即使代价是无法匹配条目，通常的上下方向键 <color:keypress>k</color> 和 <color:keypress>j</color> 依然会向上和向下滚动。</style>

<style:note>注：如果启用了 <color:keyword>quick_messages (快捷消息)</color>，那么按下未处理的按键将关闭物品列表。</style>

<topic:Travel><style:heading>旅行到物品 (Travelling to Objects)</style>

你可以通过按 <style:keypress>J</style> 或 <style:keypress>(</style> 来旅行到物品列表中当前选中的物品，这两个按键也是用于恢复被中断旅行的相同按键。（在物品列表中，无论你使用哪种按键集，这两个键的功能都是相同的。）不过，在横穿楼层旅行时要小心；你可能会一头扎进一群恶魔中，等到它们惊扰你时就为时已晚了。如果你知道该楼层大部分是空旷的，那么旅行到远处的目标是最好的选择；虽然在其他情况下也可以使用它，但请务必谨慎行事，并确保开启了 <color:keyword>check_abort (检查中断)</color> 选项 (参阅 <link:option.txt#check_abort>)。
