购物 (Shopping)

参阅 [a] 查看快捷参考 (Quick Reference)，或参阅 [b] 了解购物详情。


地表的每个城镇都包含许多商店，你可以在那里购买补给或出售不需要的物品。每
家商店都专精于某类物品：武器店 (Weapon Smiths) 
出售武器（近战和远程），而护甲店 (Armory) 出售护甲；炼金店 (Alchemist) 
出售药水和卷轴，而杂货店 (General Store) 
则出售食物、提灯和油等基础实用物品。每个城镇还有一个黑市 (Black 
Market)，在那里你可以以高昂的价格购买到通常只在地下城中才能找到的物品。

屏幕底部显示了可用命令的简要列表。只需选择相应的命令并遵循提示操作即可。
如果商店的库存有很多页，请使用 Page Up 和 Page Down 来翻页。


每位店主的资金都是有限的，所以对于你出售的物品，他们能支付的金额有一个上
限；这个金额显示在屏幕顶部的商店类型旁边。

快捷参考 (Quick Reference)
  基本命令
  p b g     从商店购买物品
  s d       向商店出售物品
  x         检视商店中的物品
  B         买空整个商店的库存
  S         花钱刷新商店库存
  R         预留商店物品以供未来购买
  q Esc     离开商店
  
  额外命令
  w         穿戴/装备物品
  t T       脱下/卸下物品
  k ^D      销毁背包中的物品
  e         显示装备栏
  i         显示物品栏
  I         检视背包中的物品
  Tab       切换物品栏/装备栏子窗口
  {         为背包中的物品铭刻
  }         擦除背包中物品的铭刻
  C         显示角色面板
  
    
    注：这些额外命令其实就是普通的游戏命令，在这里提供只是为
    了方便起见，这样你就不必离开商店去使用它们。但是，这些命
    令的按键绑定是硬编码的 
    (hard-coded)，因此你自定义的任何按键映射 (keymaps) 
    都不起作用。
  
  导航
  Page Up   滚动浏览长长的物品列表
  Page Down
  Space     与 Page Up 相同
  -         与 Page Down 相同
  
  显示选项
  ^W        切换重量显示 (参阅 [c])
  ^G        切换图形显示 (参阅 [d])
