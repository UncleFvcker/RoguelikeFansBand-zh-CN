金窝银窝不如自己的草窝 (Home, Sweet Home)

按 [a] 查看快捷参考 (Quick Reference)。

你的房屋是地表上所有常规城镇（前哨站 Outpost、莫里凡特 Morivant、阿南巴 
Anambar、萨洛斯 Thalos、特尔莫拉 Telmora 和 安格维尔 
Angwil）中都能找到的特殊建筑。在你的房屋里，你可以储存战利品以备后用；你
可以在房屋里储存的物品数量没有限制，但如果你的房子变得太乱，寻找你想要的
物品可能会成为一件苦差事。不知何故，通过某种魔法，你所有的房屋都共享同一
个库存。

g

 按 g 提取物品。按 d 存入物品。按 x 开始检视物品。                        
 按 r 开始移除 (摧毁) 物品。                                         
 按 Esc 退出。按 ? 查看帮助。                                                   

一旦物品多到屏幕无法全部显示，库存将分页列出。使用 Page Up 和 Page Down 
键进行导航。

如果你发现家里太乱，你可能想清理一下。简单的解决办法是用 g 
命令把物品拿进背包，走到外面，然后把它们丢在城镇广场上；但这乱丢垃圾，而
且还有点费事。使用 r 
命令有另一种清理方法；但是请小心，因为使用它会永久摧毁所选的物品，你不被
允许指定数量，并且你将一直处于物品移除模式，直到你按下 ESC。

请注意，你不仅可以在房屋查看你的库存，还可以通过你的角色信息面板 (C) 
查看；虽然你不能通过这种方式选择（或检视）单件物品，但这通常是在成百上千
页垃圾中定位有用物品的最快、最方便的方法，特别是当你正在整理你的装备时。
这也减少了清理的需要。

博物馆 (The Museum)


博物馆现在作为本地公房使用。它的界面和房屋相似，但库存保存在本地共享仓库
中，而不是某个单独角色的存档里。你可以把物品存入博物馆，再用同一台电脑上
的其他存档进入博物馆取出，从而在不同角色之间转移道具。


本地公房只在你的本机共享，不会连接公网服务器。由于这里的物品可以被其他存
档取走，存放重要物品前请确认你确实希望让它进入共享仓库。

快捷参考 (Quick Reference)
  基本命令
  d s       将物品从背包丢进房屋里 (存入)
  g p b     从房屋中获取物品放进背包 (提取)
  x         检视房屋中的物品
  r         从你的房屋中永久移除(摧毁)物品
  q Esc     离开房屋
  
  额外命令
  w         穿戴/挥舞一件物品
  t T       脱下一件物品
  k ^D      摧毁你背包中的物品
  e         显示装备栏
  i         显示物品栏
  I         检视你背包中的对象
  Tab       切换物品栏/装备栏副窗口
  {         为背包中的对象铭刻
  }         移除背包中对象的铭刻
  C         显示角色信息面板
  
    
    注：这些额外命令只是为了方便你而提供的普通游戏命令，这样
    你就不需要离开房屋来使用它们。然而，这些命令的按键绑定是
    硬编码的，因此你定义的任何宏或按键映射在这里都不会适用。
    
  
  导航
  Page Up   滚动浏览长长的物品列表
  Page Down
  Space     与 Page Up 相同
  -         与 Page Down 相同
  
  显示选项
  ^W        切换是否显示重量 (参阅 [b])
  ^G        切换是否显示图形 (参阅 [c])
