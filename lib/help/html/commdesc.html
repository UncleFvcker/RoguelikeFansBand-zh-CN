命令详细说明 (Command Descriptions)

以下命令说明以命令名称加上底层命令按键的形式列出。如果Roguelike 
按键集的按键与底层命令按键不同，则会在后面附加。接下来是对该命令的简短描
述，包括在需要时如何在每种按键集中指定该命令的替代方法的信息。

如果没有指定具体的重复次数，并且没有关闭 always_repeat 
选项，则几个命令（挖掘、解除陷阱、撞击、打开）会自动重复必要的次数（最多
 99 
次）。其他一些命令允许使用重复计数作为参数；例如，需要数量的命令可以使用
重复计数而不是提示询问数量，这样就可以通过按 0d 
来丢弃全部物品。提示询问数量的命令会将任何字母转换为最大合法值。

按 ? 可查看在帮助系统内部使用的命令。

物品栏命令 (Inventory Commands)

  物品栏列表 (i)
  显示当前携带但未装备的物品列表。除装备槽外，你的物品栏最多有 26 
  个槽位。相同的物品通常可以合并成一个“堆叠”，只占据一个槽位，例如下面示
  例中的 4 把火把、26 支箭和 13 份口粮；堆叠的最大数量为 99 个物品。
  
  物品栏    | 装备栏                                                    
   a) , 13 份口粮                                             13.0 lb
   b) - 魔杖 {已试}                                             0.2 lb
   c) ~ 4 把木火把 (带 2000 回合光照)                         12.0 lb
   d) ( 一件肮脏的破布                                           2.0 lb
   e) ) 一面小皮盾 {普通}                                      5.0 lb
   f) ) 一面小皮盾                                             5.0 lb
   g) | 一把左手匕首 {普通}                                    3.0 lb
   h) { 26 支箭 (3d4) (+0,+0)  
                                                              45.4 lb
  当前负重 93.4 磅 (71% 载重)。
  
  你可以在此处按 / 
  在物品栏列表、装备列表以及可能的其他物品栏（如箭袋）之间切换。
   
  装备栏列表 (e)
  使用此命令显示你的角色当前穿戴或挥舞的物品列表。你的角色通常有 13 
  个装备槽位，每个槽位对应身体的不同部位；每个槽位一次只能容纳一件物品，
  并且每个槽位只能容纳合适类型的物品。请注意，某些种族（以及大多数怪物形
  态）的身体结构与典型的类人生物不同，因此你的装备槽数量和类型可能会有所
  不同。以下是人类战士的装备列表示例：
  
   a) | 右手      : 一把阔剑 (2d6) (+0,+0)                    15.0 lb
   b) ) 左手      : 一面小皮盾 [3,+0]                          5.0 lb
   c) } 射击      : 一把短弓 (x2.50) (+0,+0)                   3.0 lb
   d)   箭袋      :
   e)   右手指环  :
   f)   左手指环  :
   g)   脖子      :
   h) ~ 光源      : 一把木火把 (带 1983 回合光照)              3.0 lb
   i) [ 身体      : 锁子甲 (-2) [14,+0]                       22.0 lb
   j)   披风      :
   k)   头部      :
   l)   手部      :
   m)   脚部      :                                                     
                                                              48.0 lb   
  
  
  每个槽位都有一个描述性标签（如“右手”），它决定了它可以接受什么类型的物
  品；例如，右手和左手可以接受武器和盾牌。而且没错，你可以挥舞两把武器或
  持有两面盾牌！大多数物品的槽位都非常直观（手套戴在手上，头盔戴在头上）
  ，但有些可能会有点令人惊讶，比如竖琴装备在“射击 (Shooting)”槽位中。

  检视物品 (I)
  
  此命令可让你详细检查物品，列出你已查明的任何物品特殊能力，说明该物品是
  否仍然具有隐藏的力量，并显示该物品的重量和估计价值。同样地，在此处可以
  使用 / 在你的背包、装备和其他相关库存之间切换。这是一个对于神器和 Ego 
  物品特别方便的命令。

  拾取物品 (g)
  此命令将拾取位于玩家同一图块上的任何物品（受 墨家明器 
  插件规则影响）。为方便起见，拾取物品不消耗时间。

  自动拾取物品 (^G)
  此命令将拾取位于玩家同一图块上的任何物品（受 墨家明器 
  插件规则影响）。此外，此命令还将自动行走到视线内并拾取其他物品。启用 
  auto_get_objects 
  选项后，任何注册为自动拾取或自动摧毁的物品都符合自动拾取的条件。如果你
  觉得这太多了（也许你喜欢在战斗后亲自打扫战场？），auto_get_ammo 
  选项只适用于刻有 =g 
  铭文的物品；这通常是你的弹药，或者可能是忍者的铁蒺藜。
  
  
  自动拾取行走将在没有更多符合条件的可见物品时停止；或者当你的背包满了时
  ；或者如果你因任何原因受到干扰（例如一只讨厌的龙醒来并决定对你喷吐）。
  自动拾取比通过物品列表行走更安全但也更局限；由于必须可见才符合条件，自
  动拾取通常只能清理单个房间，而物品列表和 H/^E 
  命令允许进行长距离的自动旅行，甚至可能深入险境（尽管当不好的事情发生时
  这些旅程也会停止）。

  丢弃物品 (d)
  
  这将把物品从你的物品栏或装备栏丢弃到地下城地板上。如果你站立的地板上已
  经有物品，或者完全拒绝物品掉落（门、楼梯、陷阱、某些地形类型如深岩浆）
  ，游戏将尝试将物品掉落到相邻的空间。如果其他方法都不起作用，物品将被丢
  弃到已经有物品的方格上；但请注意，如果附近根本没有可接受的方格，物品可
  能会完全消失。如果选择丢弃的堆叠包含多个物品，你可以指定数量。

  摧毁物品 (k) 或 (^D)
  
  这会摧毁物品栏或地下城地板上的物品。如果选择的堆叠包含多个对象，你可以
  指定数量。除了毫无价值的垃圾物品外，你必须始终在执行此命令时确认验证。
  

  穿戴/挥舞装备 (w)
  
  要穿戴或挥舞物品，请使用此命令。由于每个装备槽一次只能容纳一件物品，如
  果你将物品穿戴或挥舞到已经被占据的槽位，旧物品将首先被脱下，并且如果在
  你的物品栏中没有空间容纳它，它可能会被丢弃在地。

  脱下装备 (t) 或 (T)
  
  使用此命令脱下一件装备并将其返回你的物品栏。如果你的物品栏中没有空间容
  纳该物品，你的背包将溢出，你将在脱下该物品后将其丢弃。偶尔，你会遇到无
  法取下的被诅咒物品；这些物品通常会以某种方式惩罚你，并且只能在诅咒被移
  除后才能取下。

移动命令 (Movement Commands)

  行走 (带拾取) (;)
  在指定方向上移动一步。你要进入的方格不能被墙壁或门挡住。如果设置了 
  always_pickup 
  选项，你将拾取目标网格中的任何物品。此命令可以带有重复计数并且需要一个
  方向。你也可以使用“原版”方向键（两种按键集皆可）或“Roguelike”方向键（R
  oguelike 按键集）朝某个方向行走（它们被按键映射到此命令）。如果激活了 
  墨家明器 或简易自动摧毁器，指定的物品将被拾取或摧毁。

  行走 (翻转拾取) (-)
  
  这与正常的移动一样，只是“拾取物品”选项被反转了。换句话说，当且仅当没有
  设置 always_pickup 
  时你才会捡起东西。此命令可以带有重复计数并且需要一个方向。如果激活了 
  墨家明器 或简易自动摧毁器，指定的物品将被拾取或摧毁。
  
  这也是故意走入陷阱的命令。

  跑步 (.) 或 (,)
  
  此命令将在指定方向上移动，沿着走廊的任何弯道，直到你必须在两个方向之间
  做出“选择”或受到干扰。你可以通过设置干扰选项来配置什么会干扰你。跑步需
  要一个方向。默认情况下 Shift+DIR (方向) 被映射为 .DIR 
  (向某方向跑)，尽管使用原版按键集时在某些平台上可能会遇到技术限制。如果
  激活了 墨家明器 或简易自动摧毁器，指定的物品将被拾取或摧毁。

  上楼梯 (<)
  
  如果你站在上行楼梯或竖井上，则向上攀爬。在荒野中，除非附近有警觉的怪物
  ，否则切换全局地图模式。
  
  除非你是在茶歇模式 (coffee-break mode)下玩或激活了 
  ironman_downward（铁人下行）选项，否则每个楼层总是至少有一个上行楼梯。
  上楼梯将把你带到地表或一个新的地下城楼层。请注意，楼层是半持久的；只要
  你留在地下城中（假设存在上行楼梯），你就可以返回它们，但是当你返回地表
  时，这些楼层将永远丢失。

  下楼梯 (>)
  
  如果你站在下行楼梯或竖井上，则向下下降。在荒野中，从全局地图返回局部地
  图。
  
  
  每个楼层总是至少有两个下行楼梯，除了任务楼层（在杀死任务怪物之前没有）
  和地下城的底层。下楼梯将把你带到一个新的地下城楼层（相关说明请参阅“上
  楼梯”）。

  旅行至最近的有趣物品 (H 或 ^E)
  
  定位并自动前往最近的、满足以下条件的已知物品：未鉴定、或者未被伪鉴定、
  或者被感知为神器/Ego物品、并且未被活动的 墨家明器 
  视为不值得拾取的物品，同时已有一条已知的映射路线。这对于在清理完地下城
  楼层后，或者在杀死将战利品掉落得到处都是的高级怪物后，快速定位你忘记捡
  起或鉴定的物品非常方便。

  恢复旅行 (J) 或 (()
  
  恢复之前在到达目标前被中断的旅程。此命令也可以在怪物和物品列表中用于开
  始新的旅程；出于此目的，Roguelike 按键集也使用 J 键。

休息命令 (Resting Commands)

  原地停留 (带拾取) (,) 或 (.)
  
  在同一方格停留一回合。如果你正常拾取遇到的物品，你将捡起你站在上面的任
  何东西。此命令可以带有重复计数。你也可以使用“5”键（两种按键集）。如果
  激活了 墨家明器 或简易自动摧毁器，指定的物品将被拾取或摧毁。

  休息 (R)
  
  休息比反复原地停留对你更好，并且可以被设置为在特定时间后或满足各种条件
  时自动停止。无论如何，当发生任何干扰事件或你按下任何键时，你都会醒来。
  如果没有指定计数，此命令会要求输入一个计数；该计数默认为特殊值 
  &，这意味着“休息直到完全康复”。也接受另一个特殊值 * 
  代表“休息直到HP/SP全满”。

搜索命令 (Searching Commands)

  搜索 (s)
  
  此命令可用于定位玩家相邻空间中的隐藏陷阱和暗门。在大多数情况下，需要多
  个回合的搜索。你应始终在尝试打开箱子之前搜索它，因为它们通常有陷阱；虽
  然如果没有发现任何东西，你可能找到了一个罕见的无陷阱箱子。此命令可以带
  计数，如果你相当确定最终会找到什么，这将很有用，因为命令一旦发现任何东
  西就会停止。

  切换搜索模式 (S) 或 (#)
  
  此命令将带你进入和退出搜索模式。初次按下时，屏幕底部将显示消息“Searchi
  ng 
  (搜索中)”。现在你的每个命令都需要花费两个回合的时间，一个回合用于命令
  ，一个回合用于搜索。这意味着你在地下城中移动的时间是原来的两倍，因此消
  耗的食物也是两倍。如果你受到干扰，搜索模式将自动关闭。你也可以通过再次
  输入搜索模式命令来关闭搜索模式。

交互命令 (Alter Commands)

  挖掘隧道 (T) 或 (^T)
  挖掘隧道或采矿是一门非常有用的技艺。岩石有许多种类，硬度各异，包括：
  
    # 永久岩石         永久
    # 花岗岩           极硬
    % 石英矿脉         硬
    % 岩浆矿脉         软
    : 碎石             极软
  
  石英和岩浆矿脉有时可能含有宝藏；已知的宝藏将显示为 *
  ，但并非所有的宝藏在没有探测的情况下都是可见的。碎石有时会覆盖物品。除
  非你挥舞着重型武器或挖掘工具，或者你的力量非常高，否则挖掘隧道是很困难
  的。此命令可以带有计数，需要一个方向，并受 always_repeat 选项的影响。

  打开门或箱子 (o)
  
  要打开箱子，必须使用此命令（对于门，简单地撞上闭合的门会自动尝试打开它
  ）。如果门或箱子上了锁，你将尝试根据你的解除陷阱能力进行开锁；如果你在
  没有先解除陷阱的情况下打开带陷阱的箱子，陷阱将被触发。你可能需要多次尝
  试才能打开一扇门或箱子，很少有门会被卡住（jammed）并需要被撞开。打开命
  令可以带有计数，需要一个方向，并受 always_repeat 选项的影响。

  关门 (c)
  
  非智能和其他一些生物无法开门，所以关门非常有价值。破损的门无法关闭。把
  门撞开可能会破坏它。关门可以带有计数，需要一个方向，并受 
  always_repeat 选项的影响。

  卡住门 (j) 或 (S)
  
  许多怪物可以简单地推开关闭的门，并最终能够突破锁住的门。因此，你有偶尔
  可能想用铁蒺藜（iron 
  spikes）把门卡死。每次在门上使用铁蒺藜都会使这扇门更难被撞开，直到达到
  一定限制。较小的怪物不太能够撞开门。为了使用此命令，你必须携带铁蒺藜。
  此命令需要一个方向。

  撞门 (B) 或 (f)
  
  此命令允许你撞倒卡住的门。你的撞击能力随力量而增加。撞开门可能会让你（
  短暂地）失去平衡。卡住的门，或者被铁蒺藜卡死的门只能通过撞击来打开，并
  且如果需要，所有关闭的门都可以被撞开。撞开一扇门可能会永久性地破坏它。
  撞门可以带有计数，需要一个方向，并受 always_repeat 选项的影响。

  解除陷阱或箱子 (D)
  
  你可以尝试解除地板上或箱子上的陷阱。如果你失败了，你可能会失误并触发它
  。你只能在发现陷阱后（通过搜索命令、探测或之前触发它）才能解除它。解除
  命令可以带有计数，需要一个方向，并受 always_repeat 
  选项的影响。除非禁用了 easy_disarm (简易解除陷阱) 
  选项，否则走进已知的地板陷阱会自动尝试解除它们；如果你希望故意走入陷阱
  而不解除它，请使用 - 键。

  与相邻网格交互 (+)
  
  这个多用途的超级命令允许按单个键来选择上述任何“明显的”命令（攻击、挖掘
  、撞击、打开、解除、关闭）。在两种按键集中，Ctrl-[方向] 
  默认在宏中等同于交互-[方向]。交互命令可以带有计数，需要一个方向，并受 
  always_repeat 选项的影响。

法术和祈祷命令 (Spell and Prayer Commands)

  浏览法术列表 (b) 或 (P)
  
  所有可以施展法术的角色都可以使用此命令。使用此命令时，将显示对你可用的
  （或包含在选定魔法书中的）所有法术，以及诸如它们的等级、施放所需的法力
  值以及你是否知道该法术等信息。当你从列表中选择一个法术时，将显示该法术
  效果的简短描述。

  学习新法术或祈祷 (G)
  
  使用此命令来实际学习新法术或祈祷。当你能够学习新法术或祈祷时，屏幕的状
  态区域将出现 Study (研究) 
  字样。如果你身上有一本包含你可以学习的法术或祈祷的书，那么你可以选择研
  究那本书。如果你是牧师、圣骑士、游侠、武僧或原力行者，你的神会为你选择
  一个祈祷；其他职业可以选择他们希望研究的法术，甚至可以第二次研究同一个
  法术以降低消耗。每个法术领域都有四本书，其中有些通常只在地下城中才能找
  到。

  使用魔法法术/灵能 (m)
  
  要使用书本法术（或祈祷），你必须之前学习过该法术，并且你的物品栏中必须
  有可以从中阅读该法术的书。每个法术都有一个失败几率，开始时相当大，但随
  着等级的提高而降低。如果你没有足够的法力值来施展法术，将提示你进行确认
  。如果你决定强行施法，失败几率会大大增加，并且你可能会失去一点体质。因
  为你必须从书中阅读法术，所以施法时你不能失明或困惑，并且必须存在一些光
  源。
  
  
  对于那些不学习书本法术而是通过其他方式（通常只是通过升级）学习魔法或精
  神能力的职业，此命令可能会遵循稍微不同的规则。对于大多数（但不是全部）
  职业，如果你的角色是凡人 (Mundane) 或装备了破魔 (Anti-Magic) 
  物品，此命令将完全失效。

  使用能力 (U) 或 (O)
  一些种族和职业具有特殊的固有能力。例如，独眼巨人可以投掷巨石 (Throw 
  Boulders)，侏儒可以从危险中闪现 (Blink) 离开。战士可以学习剑舞 (Sword 
  Dancing) 
  的力量。此外，你也可能会变异并获得一种能力，例如从眼睛发射激光束 
  (Laser Beams) 的能力！如果你没有特殊能力，此命令将不起任何作用。


物品使用命令 (Object Use Commands)

  进食 (E)
  你必须定期进食以防止饥饿。当你变得饥饿时，屏幕状态栏会出现 Hungry 
  (饥饿)。如果你饿得太久，你会变得 Weak (虚弱)，然后开始 Fainting 
  (昏厥)，最终饿死。使用此命令吃掉你物品栏中的食物——尽管你也可以直接从地
  上吃东西……

  添加燃料 (F)
  
  如果你正在使用火把并且你的背包中有更多火把，或者你正在使用提灯并且你的
  背包中有油瓶，你可以使用此命令为你的光源补充燃料。火把和提灯的燃料容量
  有限；通常情况下，两瓶油将完全加满一盏提灯，两根火把将完全补充一根火把
  。

  饮用药水 (q)
  使用此命令饮用药水，它将被完全消耗，除非你是药水专精的魔法装置使 
  (Devicemaster)。效果好坏取决于具体的药水。

  阅读卷轴 (r)
  
  使用此命令阅读魔法卷轴。阅读卷轴会导致羊皮纸在魔法生效时化为灰烬。大多
  数提示输入更多信息的卷轴可以被中止（按 
  ESC）且不消耗卷轴。注意魔法是变幻莫测的，必须小心阅读卷轴；如果念错了
  咒语，什么都不会发生。
 
  激活物品 (A)
  
  你听过传闻，在地下深处有一些特殊物品，它们能让你像龙一样喷吐火焰，或者
  只需一个念头就能照亮整个房间。如果你有幸找到这样的物品，此命令将让你激
  活其特殊能力。只有已装备的物品才能被激活，并且它们通常需要一定的时间充
  能后才能再次激活。激活类似于卷轴、魔杖、法杖和魔棒的效果：都需要魔法装
  置技能 (Magic Device Skill) 并且有可能失败。

  瞄准/使用魔杖 (Wand) (a) 或 (z)
  此命令尝试使用魔杖。魔杖的效果通常是射线 (bolt)、光束 (beam)、球体 
  (ball) 或喷吐 (breath) 
  形状，因此需要瞄准；相应地，此命令需要一个方向并且可以使用目标。

  使用/挥舞法杖 (Staff) (u) 或 (Z)
  
  此命令尝试使用法杖。法杖通常具有区域效果或影响特定对象，因此不需要瞄准
  。
 
  使用/激活魔棒 (Rod) (z) 或 (a)
  
  此命令尝试使用魔棒；魔棒可以具有类似法杖或魔杖的效果。此命令可能需要一
  个方向（取决于魔棒的类型以及你是否了解它的类型）并且可以使用目标。

  铭刻物品 (Z) 或 ({)
  
  此命令在物品上铭刻一串字符。铭刻显示在物品描述后的大括号内。铭文仅限于
  特定的物品（或堆叠），并且不会自动转移到所有类似物品上。某些铭文向游戏
  提供了如何处理或标记该物品的指令；有关更多信息，请参阅 [a]。

  移除铭刻 (})
  此命令移除物品上的铭文。此命令对游戏本身添加的“虚假”铭刻没有影响。

投射与远程武器命令 (Throwing and Missile Weapons)

  射击/投掷 (f) 或 (t)
  
  发射适合你当前远程武器的弹药，前提是你装备了远程武器。投石索只能发射石
  子和铁弹；弓只能发射箭；弩只能发射弩箭。弹药通常从你的箭袋发射，但也可
  以直接从你的物品栏甚至从地板上发射。此命令需要一个方向，并可以接受一个
  目标。

  投掷物品 (v)
  
  你可以投掷你的角色携带的任何物品。物品越轻，你扔得越远。一次只能投掷堆
  叠中的一个物品。抛出的物品可能会或可能不会损坏怪物。投掷物品可能会打破
  它。把药水砸在墙上总会打破它，可能造成溅射伤害。此命令需要一个方向，并
  可以接受一个目标。

  设置目标 (*)
  
  此命令允许你将法术、箭矢等瞄准特定的怪物或网格，即使它们不在八个指南针
  方向之一；它在方向提示的内部和外部都有效。此命令也可以替代四处观察 
  (look) 命令，如果你的观察目标是个怪物，这尤其方便。
  
  
  瞄准提示提供了许多选项。提供的第一个目标将是视线内最近的怪物，视线内的
  其他怪物作为备选目标。使用方向键在怪物之间循环切换；+ 和 - 
  也起作用，但不太直观。
  
  如果你想瞄准（或检查）一个没有怪物的方格，请按 p 或 
  o，切换到方向键只需移动光标的模式。（如果没有敌对怪物在视线内，将自动
  进入此模式。o 和 p 除了在一个方面之外是相同的：按 p 
  会将光标移动到你自己的位置）。随时可以按 5、t 或 . 接受当前目标，按 q 
  或 ESC 退出瞄准模式，按 r 回忆有关当前瞄准怪物的知识背景，按 x 
  查看目标方格上的所有物品，以及按 J/( 
  选择一个方格作为旅行目标。详情请参阅 [b]。

   (direct) 
  路径，这可能不一定是你想要的路径。有时，通过巧妙地选择一个侧面位置作为
  目标，你也许能瞄准不在直接视线内的方格：

观察命令 (Looking Commands)

  显示怪物列表 (Y) 或 ([)
  此命令显示所有当前可见怪物的列表。这是判断附近是否有 Unique 
  或其他具有威胁性怪物的快捷方法，最常用于召唤或探测之后。

  显示物品列表 (])
  
  此命令显示楼层上所有已知物品的列表，以及其他有趣的附近特征，如商店或地
  下城入口。在物品列表处于活动状态时按 S 
  会切换列表中是否包含楼梯。物品列表是定位楼梯、神器、未鉴定物品和楼层上
  其他有趣目标的便捷方法，并且通过向下滚动到目标并按 J 
  就可以直接前往该目标。

  全屏地图 (M)
  
  此命令将显示整个地下城的地图，按比例缩放以适合单个屏幕。由于比例的原因
  ，只有主要的地形特征可见，甚至一些重要物品也不会显示在地图上。这对于快
  速识别地下城的未探索区域或确定你相对于地下城边缘的位置特别有用。此命令
  不消耗时间。

  活动，指定物品可以显示在此地图上；但这是一个过时的功能，因为直接使用 
  ] 物品列表总是更方便。

  在地图上定位玩家 (L) 或 (W)
  此命令让你滚动你的地图，查看当前地下城楼层的所有扇区，直到你按 
  ESC，此时地图将根据需要重新以玩家为中心。要滚动地图，只需按任何“方向”
  键即可。顶行将显示扇区位置，以及与你当前扇区的偏移量。此命令不消耗时间
  。

  重新居中视口 (^V)
  
  此命令使视口重新以玩家为中心；当你在移动时，游戏确实会自动调整视口，但
  仅当你接近边缘时。此命令是上面讨论的在地图上定位玩家命令的更快替代品，
  尽管有些玩家更进一步，直接激活 center_player 选项 ([c])。

  四处观察 (l) 或 (x)
  
  此命令用于观察附近的怪物（以确定其类型和生命值）和物品（以确定其类型）
  。它也用于查明怪物下方有什么物品（如果有的话），以及怪物当前是否在墙内
  。此命令不消耗时间。
  
  此命令与 * 
  瞄准命令非常相似，并且瞄准提示的大多数特殊键也可以在观察提示中起作用。
  使用 * 
  而不是此命令通常很方便，特别是如果你想观察一个怪物的话；另一方面，l/x 
  更适合用于检查城镇建筑。

  存档与退出命令 (Saving and Exiting Commands)
  
    保存并退出 (^X)
    
    要保存你的游戏以便以后返回游玩，请使用此命令。如果游戏由于系统错误而
    崩溃，也（有望）会生成保存文件。在你死亡后，你可以使用你的保存文件以
    相同的选项和背景知识再次开始游戏。
  
    保存 (^S)
    
    此命令保存游戏但不退出。如果你经常偏执地担心电脑在你玩游戏时崩溃，请
    经常使用这个命令。
  
    退出并自杀 (Q)
    
    杀死你的角色并退出游戏。系统会提示你以确保你真的想这么做，然后要求你
    验证该选择。请注意，死掉的角色就永远死掉了。
  
  用户偏好命令 (User Preference Commands)
  
  有关用户偏好命令的描述以及此功能的详细说明，请参见 [d]。
  
    与选项交互 (=)
    
    允许你与选项交互。请注意，使用作弊选项可能会将你的保存文件标记为不适
    合高分榜。
  
    与宏交互 (@)
    
    允许你与宏交互。你可以从用户偏好文件中加载或保存宏，创建各种类型的宏
    ，或定义按键映射。
  
    与视觉效果交互 (%)
    
    允许你与视觉效果交互。你可以从用户偏好文件中加载或保存视觉效果，或修
    改怪物、对象和地形特征的属性/字符映射。
  
    与颜色交互 (&)
    允许用户与颜色进行交互。此命令仅在某些系统上有效。
   
    重载 墨家明器 偏好       ($)
    此命令将为 墨家明器 重新加载 "pickpref.prf" 和 
    "pickpref-<<playername>>.prf"；仅当你在游戏期间使用外部文本编辑器编
    辑过这些文件时才需要使用。
  
    编辑 墨家明器 偏好       (_)
    允许你编辑拾取偏好，并在 墨家明器 尚未激活时将其激活。ESC 
    键将在此编辑器中打开菜单。你可以从命令菜单中查看和使用所有编辑命令。
    按 ^Q 或 ^W 退出编辑器。
  
  帮助命令 (Help)
  
    帮助 (?)
    
    调出在线帮助系统。在某些情况下（如在帮助系统内），它可能会调出特定的
    帮助页面。
  
    查询符号含义 (/)
    使用此命令查找某个字符代表什么；例如，通过按下 /.，你可以发现 . 
    符号代表一个地板位置。注意如果你希望看到关于附近特定怪物的信息，你应
    该使用 * 命令而不是此命令。
    
    
    如果你对怪物符号使用查询符号命令，你将被提示该符号对应的物种，并获得
    你见过的所有该物种怪物的列表。类似的信息可通过 ~ 知识菜单获取。
    
    查询符号有四个特殊的组合键：Ctrl-A 回忆所有已知怪物，Ctrl-U 
    回忆所有已知 Unique，Ctrl-N 回忆所有已知非 Unique，以及 Ctrl-M 
    回忆所有匹配你输入字符串的已知怪物。按 k 
    按击杀次数对这些列表排序，按 p 按怪物的原生深度对它们排序，按 ESC 
    退出。
   
    游戏版本 (V)
    
    此命令将告诉你正在使用的游戏版本，以及地图的当前大小。有关更多信息，
    请参见 [e]。
  
  额外命令 (Extra Commands)
  
    重复上次命令 (n) 或 (X)
    
    此命令重复上次执行的命令及其所有参数。对于向特定敌人反复射击或施展相
    同法术非常有用。
  
    录制命令 (")
    你输入的命令可以记录到任何正常的 ASCII 
    字符寄存器中。实际上，你手动输入的每个命令都会自动记录到 . 
    寄存器中。当你重复上一个命令时，你实际上是在回放这个特殊的 . 
    寄存器的内容。但是你也可以将下一个命令记录到你选择的寄存器中，并在稍
    后手动回放它；例如，命令序列 "Fmae*t 将命令 mae*t 除了记录到 . 
    寄存器外，还记录到了 F 寄存器。你稍后可以用 'F 回放此命令，而用 '. 
    或 n 回放仅在下一个回合有效。
  
    回放录制的命令 (')
    如上所述，你可以使用 " 命令录制命令，并在稍后使用 ' 
    命令回放它们。命令存储在寄存器中，每个寄存器用一个 ASCII 
    字符寻址。例如，如果你使用 "t<cmd> 
    录制了一个命令，那么你稍后可以通过按 't 
    来回放它。这些命令寄存器是临时的，并且在每次重启游戏时都会被清除。要
    查看你记录的命令，选择 ' 寄存器（即从顶层按 
    ''）。然后你可以输入你想使用的寄存器（例如 ''t 的工作原理与 't 
    相同，只是在你按下第二个 ' 
    之后会向你显示所有已记录命令的寄存器内容）。
    
    回放命令很少使用，因为宏和按键映射 ([f]) 通常更佳。
  
    切换选择窗口 (^I)
    
    在物品栏和装备栏之间切换副窗口显示。此命令仅当你启用了物品栏/装备栏
    副窗口时才适用。
  
    重绘屏幕 (^R)
    此命令适应全局选项中的各种更改，并重绘所有窗口。当你在 angband.live 
    上玩耍并且有人开始观看你的游戏时，使用此命令是一种礼貌；如果你觉得有
    图形错误或者游戏没有正确响应你所做的视觉改变，也请使用此命令。有疑问
    时，就用它。
          
    创建屏幕截图 ())
    
    此命令创建当前屏幕的文档，然后在游戏的内部文件查看器中向你显示该文档
    。然后你可以使用 | 命令将文档另存为纯文本或 html。要创建 html 
    文件，只需给文件添加 html 
    扩展名。注意如果你在启用了图形的情况下进行游戏，那么此命令将在创建屏
    幕截图之前尝试将其转换为正常的文本显示。
  
    指挥宠物 (p)
    如果你有宠物，请使用此命令下达命令或切换宠物选项。详情请参见 [g]。
