宽度始终是 66 格的倍数（最大不超过 198），高度始终是 22

(big)，这意味着它们通常总是生成尽可能大的地牢层（198x66），而“迷宫 
(Labyrinth)”、“兽穴 (Warrens)”和“矿坑 (Mine)”则被标记为最小 
(smallest)，这意味着它们总是生成尽可能小的地牢层（66x22）。某些地牢（尤
其是主地牢，安格班 Angband）遵循茶歇模式 
(coffee-break)的地牢生成规则，这意味着地牢层的平均大小会随着深度的增加而
增加；在浅层会非常频繁地创建小型地牢，但当玩家到达深层地牢时，这些“小型”
地牢层的面积开始接近大型地牢层，通常为 198x44 或 
132x66。其余的地牢遵循标准生成 (standard generation)规则，这意味着有 
1/5 的几率生成偏小的地牢层（排除了极小的地牢层），有 4/5 的几率生成 
198x66 的大型地牢层。

修改 地牢层大小 (level size) 
这个出生/建卡选项，允许你除了遵循上述概述的标准方法外，还能从 10 
种略有不同的生成地牢层大小的方法中进行选择。为了清晰起见，本说明使用 
1x1 来指代一个 66x22 的区块；因此，例如 2x3 意味着地牢层宽 132 格、高 
66 格。所有选项都将 1x2 和 2x1（或 2x3 和 3x2 
等）同等对待，尽管实际生成的地牢层比例不同（66x44 / 132x22）。除了“庞大 
(Huge)”选项外，这些选项都不会对三个“最小”地牢（它们总是生成微小地牢层）
或竞技场 (Arena) 地牢（它总是生成庞大地牢层）产生任何影响。 
当且仅当启用 no_big_dungeons (无大型地牢) 
选项时，这些选项才会对除竞技场外的其他“大型”地牢产生影响。

庞大 (Huge)
   此设置在所有地牢中强制生成 3x3 
   的大型地牢层，甚至包括那些通常只会生成 1x1 地牢层的地牢。

庞大，保留微型 (Huge - respect tiny)
   此设置允许被标记为 最小 (smallest) 的地牢像往常一样生成 1x1 
   地牢层，但强制所有其他地牢层为 3x3 大小。

正常 (Normal)
   默认设置，详情如上文所述。

茶歇模式地牢生成，允许大型 (Coffee-break level generation - allow big)
   
   开启此设置后，通常遵循标准生成或总是生成大型地牢层的地牢，将改为遵循
   茶歇模式算法。游戏中偶尔仍然会创建出大型地牢层，因为茶歇模式的地牢生
   成允许偶尔出现大型地牢，特别是在深层地牢中。

茶歇模式地牢生成，从不大型 (Coffee-break level generation - never big)
   此设置与上面相同，只是永远不会生成大型地牢层。

中等 (Medium)
   开启此设置后，所有地牢层的大小将在 1x2 和 2x3 
   之间。在“大型”地牢中应用 2x2 
   的最小尺寸限制。地牢深度对地牢层大小没有影响。

速溶模式地牢生成，允许大型 (Instant-coffee level generation - allow 
big)
   此设置几乎完全根据深度来决定地牢层大小，在游戏前期生成 1x1 
   的微型地牢层，在游戏后期生成 3x3 
   的大型地牢层。地牢层大小存在一定的随机性，但比大多数其他设置的随机性
   要小。在大型地牢中，用于计算生成比例的深度数值会被略微提高。

速溶模式地牢生成，从不大型 (Instant-coffee level generation - never 
big)
   此设置与上面类似，但地牢层尺寸稍小，最大为 2x3 而不是 3x3。

小型，经典 (Small - classic)
   开启此设置后，所有地牢层的大小将在 1x1 和 2x2/1x3 
   之间，但在大型地牢中它们是 2x2 或 
   2x3。（如果在大型地牢中不启用小型层，这就是旧版本 PosChengband 
   中“经典”的小型地牢层选项。）

小型，茶歇风格 (Small - coffee-style)
   此设置也会生成大小在 1x1 到 2x2/1x3 
   之间的地牢层，但不再是完全随机的：地牢层尺寸在较浅的深度时更有可能落
   在这个范围的较小端，而在较深的深度或在大型地牢中，更有可能落在较大端
   。

极小 (Very small)
   此设置主要生成 1x2 的地牢层，但也允许生成 
   1x1（大型地牢除外），并偶尔允许 2x2/1x3（特别是在大型地牢中）。

超小 (Extremely small)
   此设置在大型地牢中只生成 1x2 的地牢层，而在所有其他地牢中只生成 1x2 
   或 1x1 的地牢层。

微型 (Tiny)
   
   看来你真的很喜欢微型地牢！开启此设置后，每个地牢层（即使是在“大型”地
   牢中）都被保证是 1x1 的微型大小。
